# Speech To Game

음성 인식된 단어들을 타이핑해서 없애자

## Requirements

- Chromium Browser (Firefox unsupported)
- Microphone (Need Auth)

- CSS framework [NES.css](https://nostalgic-css.github.io/NES.css/)
